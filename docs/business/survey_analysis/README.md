# Survey Analysis Outputs

This folder records the anonymised aggregate analysis of the current 60-response NeTrix academic communication survey. The raw Excel workbook is not stored here because it contains respondent names and email addresses.

## Contents

| Path | Purpose |
| --- | --- |
| `SURVEY_ANALYSIS_REPORT.md` | Exploratory data analysis report for the 60-response survey. |
| `METHODOLOGY.md` | Data handling, cleaning, statistical-inference position, and privacy notes. |
| `figures/` | SVG visualisations generated from aggregate survey results. |
| `tables/` | CSV summary tables with counts, percentages, and selected illustrative intervals. |

## Evidence Position

The survey should be used as exploratory discovery evidence. It supports the NeTrix problem framing and indicates promising demand signals, but it should not be presented as representative proof of the full UNNC population or even the full Math / CS / EEE first wedge without further targeted validation.

## Recommended Use In The Business Plan

Use this folder to support survey-related analysis in the business plan. The figures and tables are intentionally descriptive rather than prescriptive: they provide aggregate evidence, while leaving the Biz team space to decide how each finding should be incorporated into the final narrative.

## Figure Set

The visualisations are designed as a compact EDA set rather than a single chart type repeated across variables. The set includes a lollipop chart, a sample-composition snapshot, a Likert-style stacked distribution, a dot plot, a participation funnel, a study-buddy flow figure, and a radar chart for AI-tool evaluation dimensions.
